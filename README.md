# peckworks-codebase-rag

A generic, code-aware RAG (Retrieval-Augmented Generation) tool: point it at any
repo, ask natural-language questions, get grounded, cited answers, or an honest
refusal. One tool, many indexes (one Qdrant collection per repo). Evolves the
proven pipeline from `peckworks-rag-lab` to be code-aware and reusable.

`docs/superpowers/specs/2026-06-20-codebase-rag-design.md`. Implementation pending.
